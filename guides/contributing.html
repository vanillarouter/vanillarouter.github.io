Contributing

About Contributing

When it comes to open source projects, there are so many types of contributions that can be made. Please feel free to contribute and provide new updates.

Some topics you can contribute:

  • Edit codes anywhere in the project.
  • Update or edit the documentation page.

What should you do?

If you want to make an update or edit on the project or the documentation page, you can send a pull request after the changes by forking the github repos we shared in the links below.

Vanilla Router Dom: https://github.com/can-erturk/vanilla-router-dom
Documentation: https://github.com/can-erturk/can-erturk.github.io/tree/main/docs/vanilla-router
